Embedding links in HTML

Links are embedded using Anchor tag : <a>

Target Values

Default or _self :
Target value = "Default" or "_self", then the link will open in current tab
Open Google
_blank
Target Value = "blank" ,then the link will open in different tab generally to the next tab
Open facebook
_parent
target Value = "parent" , then link will open in the parent frame
Open Quora
_top
Target value = "top" , then link will open in the full body of window
Open wikipedia
